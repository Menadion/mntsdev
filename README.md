# !IMPORTANT!
- XAMPP Control Panel (Apache & MySQL started)
- Folder Location: "C:\xampp\htdocs\"

### How to open:
1. Run XAMPP Control Panel
2. Start Apache & MySQL
3. Open Browser
4. Go to http://127.0.0.1/
5. Click on the folder named 'mntsdev'

### For new Downloads:
1. Go to "xampp/htdocs/"
2. For new downloads, delete all files there
3. Put the mntsdev folder
